# Hub2gether — Version blanche

Le site a été adapté en thème clair (fond blanc + texte sombre).

## PNG à préparer pour remplacer les images du site

Déposez vos fichiers PNG dans le projet (par exemple dans un dossier `assets/`) et utilisez ces noms pour rester organisé :

1. `preview-dashboard.png` → Aperçu slide 1 (Dashboard admin).
2. `preview-mobile-matches.png` → Aperçu slide 2 (Écran mobile matchs).
3. `preview-groups.png` → Aperçu slide 3 (Écran groupes).
4. `preview-leaderboard.png` → Aperçu slide 4 (Écran classement).
5. `preview-custom-screenshot.png` → Zone “Votre screenshot” (uploader libre).

## Formats recommandés

- **Format** : PNG
- **Dimensions idéales** :
  - Slides larges : `1600x1000`
  - Slide mobile : `900x1600`
  - Screenshot libre : minimum `1200px` de large
- **Poids recommandé** : `< 1.5 MB` par image pour de bonnes performances.

## Comment les changer sur le site

Le site permet déjà de remplacer les images via les boutons **“Remplacer”** et la zone **“Votre screenshot”**. Vous pouvez donc tester immédiatement vos PNG depuis l'interface.
